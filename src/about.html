About

This site is a compact, static demo created to capture the visual spirit of 80s synthwave: neon gradients, pixel grids, and bold monospace typography. It was built as a simple, easily-extensible template for personal pages, portfolios, and small projects.

Design goals

  • Fast and lightweight: minimal dependencies, static HTML/CSS/JS.
  • Retro aesthetic with modern accessibility and responsiveness.
  • Easy to customize: swap colors, content, or add pages.

Tech & structure

Built with plain HTML and TailwindCSS, plus a tiny JS terminal demo. The page structure uses a centered card, a two-column grid with a terminal sidebar, and consistent utility classes for predictable styling.

License & usage

Feel free to copy, adapt, and use this template for your personal projects. No special build tools required — just edit the HTML and deploy.
